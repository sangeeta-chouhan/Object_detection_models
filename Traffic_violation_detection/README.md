# 🐱🐶 Cat vs Dog Classification using VGG19

## 📌 Project Overview
This project uses VGG19 (pretrained model) with fine-tuning to classify images of cats and dogs.

## 🧠 Model Details
- Model: VGG19
- Technique: Transfer Learning + Fine-Tuning
- Dataset: Cats vs Dogs

## ⚙️ Steps
1. Load pretrained VGG19
2. Freeze base layers
3. Fine-tune last layers
4. Train model

## 📊 Results
- Training Accuracy: 0.9785%
- Validation Accuracy: 0.9767%
